Fitur Umum Sistem

Navigasi dan Struktur Menu

Semua peran memiliki layout utama yang sama: sidebar di kiri untuk navigasi dan topbar di atas untuk akses cepat ke notifikasi dan akun pengguna.

Screenshot 1: Layout Umum (Sidebar & Topbar)

Sistem Notifikasi

TA Cloud menggunakan sistem notifikasi realtime. Anda akan menerima notifikasi untuk aktivitas penting seperti:

  • Pengajuan proposal baru
  • Permintaan sidang baru
  • Catatan bimbingan baru
  • Permohonan buka kunci nilai
Screenshot 2: Dropdown Notifikasi

Penanganan Dokumen

Dokumen diunggah dalam bentuk versi (versioning). Artinya, setiap revisi tersimpan sebagai catatan terpisah dan tidak menimpa versi sebelumnya. Dokumen dapat dipreview dalam modal PDF viewer sebelum diunduh.

Screenshot 3: Preview Dokumen PDF

Mekanisme Lock / Unlock Nilai

Setelah dosen menyelesaikan input nilai, sistem dapat mengunci nilai tersebut untuk mencegah perubahan tidak sah. Jika perlu perubahan, dosen harus mengajukan permintaan buka kunci. Kaprodi kemudian akan meninjau dan menyetujui permintaan tersebut.

Catatan: Proses buka kunci nilai adalah alur resmi dan akan meninggalkan jejak aktivitas di sistem.

Batasan Sistem Saat Ini

Dokumen ini mencatat fitur yang belum aktif penuh atau masih bersifat prototipe. Informasi ini penting untuk mengelola ekspektasi pengguna.
  • Final Submission Mahasiswa: Route dan controller sudah aktif, tetapi visibilitas menu tetap bergantung pada fase dan status dokumen.
  • Ekspor / Rekap Skripsi: Backend ekspor CSV/PDF belum tersedia penuh.
  • Publish ke Library: Halaman library sudah ada, tetapi backend untuk memindahkan skripsi final ke library belum lengkap.
  • Phase Board & Keputusan Akhir: Halaman tersedia sebagai tampilan, tetapi automation/workflow inti masih belum selesai.
  • RBAC Lanjutan: Sistem masih berbasis role dasar; Program Admin dan Super Admin belum diterapkan penuh.
  • Workflow per Program Studi: Konfigurasi fase berbeda antar prodi belum tersedia.
  • API / Token Auth: Google login sudah aktif, tetapi layer API/mobile token auth belum menjadi jalur aktif.

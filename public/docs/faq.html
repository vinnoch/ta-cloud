Pertanyaan yang Sering Diajukan (FAQ)

Jika masalah Anda tidak tercantum di sini, hubungi admin sistem atau Kaprodi di program studi Anda.
Butuh visual alur? Buka UML Activity Diagram untuk melihat alur skripsi end-to-end dalam format visual yang lebih formal.

Mengapa saya tidak bisa mengakses halaman tertentu?

Sistem akan memeriksa peran akun Anda. Jika akun Anda tidak memiliki hak akses ke halaman tersebut, sistem akan menolak akses dengan status 403 Forbidden.

Mengapa menu Final Submission tidak muncul?

Fitur Final Submission sudah aktif, tetapi tampilannya di akun Mahasiswa tetap bergantung pada fase skripsi, status review, dan kelengkapan dokumen akhir. Jika menu belum terlihat, periksa status skripsi Anda di halaman detail atau hubungi Kaprodi.

Apakah saya bisa login dengan akun Google?

Ya. TA Cloud sekarang mendukung login Google. Jika kampus Anda mengaktifkan domain/email yang sesuai, Anda dapat memakai tombol Masuk dengan Google pada halaman login.

Mengapa nilai saya terkunci?

Nilai dikunci untuk mencegah perubahan setelah finalisasi. Jika ada kesalahan input, dosen penguji atau pembimbing dapat mengajukan permohonan buka kunci kepada Kaprodi.

Bagaimana jika saya salah upload dokumen revisi?

Mahasiswa dapat menghapus file revisi yang telah diunggah dan menggantinya dengan dokumen yang benar, sesuai dengan hak akses yang tersedia di halaman detail skripsi.

Mengapa saya tidak bisa membuat data Non-Skripsi?

Fitur ini hanya tersedia jika skripsi Anda didaftarkan dengan tipe non_skripsi. Jika Anda tidak melihat menunya, periksa tipe skripsi Anda di halaman detail skripsi.

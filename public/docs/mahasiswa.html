Ringkasan Peran

Mahasiswa memakai sistem untuk mengajukan skripsi, mengunggah proposal dan revisi, membaca catatan bimbingan, melihat nilai, dan jika berlaku mengelola data non-skripsi.

Menu Skripsi

Menu inti Mahasiswa ada di area skripsi.

  • Search: pencarian data skripsi milik sendiri.
  • Index: daftar / ringkasan skripsi aktif.
  • Create: membuat pengajuan skripsi baru.
  • Show: melihat detail skripsi.
  • Edit: mengubah data skripsi.
  • Destroy: menghapus skripsi bila masih diizinkan.

Dokumen Proposal & Revisi

Mahasiswa dapat mengunggah proposal awal dan dokumen revisi lanjutan.

  1. Buka detail skripsi.
  2. Pilih menu atau tombol upload proposal / dokumen.
  3. Pilih file yang sesuai.
  4. Unggah file. Sistem akan menyimpan versi dokumen.
  5. Jika diperlukan, buka file proposal melalui menu proposal file.
  6. Jika salah unggah, hapus dokumen yang relevan bila fitur tersedia pada fase tersebut.
Sistem menyimpan versi dokumen. Gunakan nama file yang rapi agar histori mudah dibaca.

Menu Bimbingan

Mahasiswa dapat melihat daftar bimbingan, menindaklanjuti revisi, dan mengekspor riwayat bimbingan.

  • Index: seluruh sesi bimbingan untuk skripsi terkait.
  • Export CSV: ekspor daftar bimbingan ke format CSV.
  • Export PDF: tampilan cetak riwayat bimbingan.
  • Update: kirim tanggapan revisi / unggah file revisi ke sesi tertentu.
  • Destroy Revision: hapus file revisi yang terkait dengan sesi bimbingan jika salah unggah.

Final Submission

TA Cloud memiliki endpoint final submission berdasarkan event tertentu.

  1. Buka halaman final submission jika tersedia dari detail skripsi.
  2. Pilih event submission yang sesuai.
  3. Unggah atau kirim dokumen akhir sesuai instruksi.
Final submission tersedia melalui route aktif, tetapi kemunculan menu/entry point di UI tetap bergantung pada event dan fase skripsi yang sedang berjalan.

Menu Nilai

Menu nilai menampilkan hasil penilaian yang sudah tercatat untuk skripsi Anda.

  • Buka detail skripsi.
  • Pilih menu Nilai.
  • Tinjau item nilai, reviewer, dan hasil total bila tersedia.

Menu Non-Skripsi

Jika jenis tugas akhir Anda adalah non_skripsi, maka menu ini dipakai untuk menyimpan data tambahan.

  • Index: melihat semua record non-skripsi milik sendiri.
  • Create: membuat record baru.
  • Show: melihat detail record.
  • Edit / Update: memperbaiki informasi.
  • Destroy: menghapus record non-skripsi.
Field umum non-skripsi: ringkasan, abstrak, laporan, publikasi, dan nilai akhir.
Screenshot 5: Form non-skripsi

Ekspor / Arsip Pribadi

Pada sisi Mahasiswa, fitur arsip yang benar-benar tersedia saat ini paling jelas ada di menu bimbingan:

  • Export CSV: mengekspor daftar bimbingan ke file CSV.
  • Export PDF: menghasilkan tampilan PDF/print untuk riwayat bimbingan.
  • Proposal File: membuka file proposal yang sudah diunggah.
Saran: gunakan ekspor PDF bimbingan sebagai arsip progres formal, lalu simpan dokumen proposal dan revisi penting secara terpisah jika dibutuhkan.

Selamat Datang di TA Cloud

Panduan pengguna (User Guide) ini dirancang untuk membantu Anda memahami dan menggunakan sistem TA Cloud. TA Cloud adalah platform manajemen tugas akhir yang mendigitalisasi proses pengajuan skripsi, bimbingan, sidang, hingga arsip dokumen final.

Catatan: Panduan ini dibuat berdasarkan rilis sistem per Mei 2026. Beberapa fitur mungkin masih dalam tahap pengembangan (lihat Batasan Sistem).

Pilih Peran Anda

Pilih salah satu kartu di bawah ini untuk melihat panduan khusus sesuai dengan akses yang Anda miliki di sistem TA Cloud.

Kaprodi
Panduan Kaprodi
Kelola master data, plotting dosen pembimbing/penguji, dan pantau seluruh skripsi aktif di program studi.
 Dosen
Panduan Dosen
Lakukan bimbingan, berikan catatan revisi, ajukan permohonan sidang, dan input nilai akhir.
 Mahasiswa
Panduan Mahasiswa
Ajukan proposal, unggah dokumen revisi, pantau catatan bimbingan, dan lihat hasil nilai sidang.

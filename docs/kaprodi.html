Panduan Kaprodi

Kaprodi

Ringkasan Peran

Sebagai Kaprodi, Anda memiliki akses penuh ke sistem. Tugas utama Anda meliputi pengelolaan data master prodi, dosen, mahasiswa, hingga pemantauan alur skripsi mahasiswa mulai dari proposal hingga kelulusan.

Pengelolaan Master Data

Menu ini tersedia pada sidebar untuk mengelola entitas berikut:

  • Program Studi: Kelola profil prodi.
  • Dosen/Mahasiswa: Impor via CSV untuk efisiensi di awal periode.
  • Tahun Akademik & Periode: Menentukan masa aktif skripsi.
  • Format Nilai: Mengatur rubrik penilaian sidang.
Screenshot 1: Form Impor Data (Dosen/Mahasiswa)

Manajemen Skripsi

Pada dashboard Kaprodi, Anda dapat melihat statistik skripsi berdasarkan fase. Pada halaman detail skripsi, Anda dapat:

  • Menetapkan Dosen Pembimbing dan Penguji (plot reviewer).
  • Menyetujui atau menolak Permohonan Sidang yang diajukan dosen.
Screenshot 2: Phase Board Monitoring

Penilaian & Unlock Nilai

Setelah sidang selesai, Kaprodi dapat memantau rekap nilai akhir. Jika dosen perlu memperbaiki nilai karena kesalahan input setelah terkunci, Anda dapat menggunakan fitur Request Buka Kunci.

Catatan: Fitur buka kunci nilai memerlukan verifikasi manual dari Kaprodi setelah dosen mengajukan permintaan.
Screenshot 3: List Nilai Final & Approval Buka Kunci

Versi Cetak / Arsip PDF

Untuk kebutuhan rapat atau arsip manual, halaman Kaprodi dapat dicetak langsung dari browser. Gunakan mode cetak browser setelah membuka halaman yang relevan, misalnya monitoring skripsi atau daftar nilai.

Saran: gunakan tampilan desktop sebelum mencetak agar tabel dan ringkasan tetap rapi.

Panduan Kaprodi

Kaprodi

Ringkasan Peran

Kaprodi bertugas mengelola data akademik inti, memantau seluruh skripsi aktif, memproses approval penting, dan menjaga kualitas dokumen serta penilaian akhir.

Dashboard

Dashboard menampilkan ringkasan jumlah skripsi aktif, status assignment reviewer, permohonan sidang, dan distribusi fase skripsi.

Screenshot 1: Dashboard Kaprodi

Pengelolaan Master Data

  • Dosen
  • Mahasiswa
  • Tahun Akademik
  • Periode
  • Format Penilaian
  • Import CSV Dosen / Mahasiswa
Screenshot 2: Form CRUD / Import Master Data

Monitoring Skripsi

Pada halaman skripsi, Kaprodi dapat melihat detail mahasiswa, meninjau proposal, bimbingan, logbook, dan dokumen, sekaligus menetapkan pembimbing, penguji, status, dan jadwal sidang.

Screenshot 3: Halaman Monitoring Skripsi

Approval & Review

Workflow approval aktif meliputi approve/reject proposal submission, approve/reject sidang request, dan approve final review.

Catatan: phase board dan keputusan akhir masih tersedia sebagai workspace monitoring. Automation penuh belum selesai.

Nilai & Unlock

Kaprodi dapat melihat rekap nilai akhir dan memproses permintaan buka kunci nilai dari dosen jika diperlukan koreksi setelah finalisasi.

Screenshot 4: Rekap Nilai dan Approval Unlock

Template Dokumen Final

Modul template dokumen final sudah aktif. Kaprodi dapat membuat, mengubah, menggandakan, menghubungkan template ke periode, dan melepas periode dari template terkait.

Screenshot 5: Halaman Template Dokumen Final

Versi Cetak

Gunakan fitur cetak browser saat membutuhkan arsip rapat, rekap nilai, atau monitoring skripsi dalam format PDF.

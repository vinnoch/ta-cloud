Panduan Mahasiswa

Mahasiswa

Ringkasan Peran

Mahasiswa memakai TA Cloud untuk mengajukan skripsi, mengunggah proposal dan revisi, memantau bimbingan, mengirim final submission, dan melihat hasil penilaian.

Skripsi Saya

Menu ini menjadi pusat aktivitas tugas akhir Anda. Di sini Anda dapat membuat skripsi baru, memperbarui data, dan memantau fase saat ini.

Screenshot 1: Dashboard / Detail Skripsi Mahasiswa

Proposal & Dokumen

Mahasiswa dapat mengunggah proposal awal dan dokumen revisi berikutnya. Semua file disimpan sebagai versi terpisah agar histori tetap terjaga.

Screenshot 2: Upload Proposal dan Dokumen Revisi

Bimbingan

Riwayat bimbingan menampilkan catatan dosen, status revisi, dan dokumen terkait. Mahasiswa juga dapat mengekspor riwayat bimbingan ke CSV atau PDF.

Screenshot 3: Riwayat Bimbingan + Export CSV/PDF

Final Submission

Final submission sudah aktif. Mahasiswa dapat membuka form final submission sesuai event yang tersedia dan mengirim dokumen final melalui route aktif di sistem.

Pastikan seluruh revisi dan dokumen pendukung telah sesuai sebelum melakukan final submission.
Screenshot 4: Form Final Submission

Nilai

Setelah penilaian selesai, Mahasiswa dapat melihat hasil nilai pada menu nilai di detail skripsi.

Non-Skripsi

Jika tipe tugas akhir adalah non-skripsi, Mahasiswa dapat mengelola abstrak, laporan, dan tautan publikasi melalui modul non-skripsi.

Screenshot 5: Form Non-Skripsi

Versi Cetak

Mahasiswa dapat mencetak halaman riwayat bimbingan atau detail skripsi untuk arsip pribadi atau kebutuhan konsultasi.
